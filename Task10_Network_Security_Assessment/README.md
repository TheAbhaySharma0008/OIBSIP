# Task 10 - Network Security Assessment

## Objective
Perform a basic network security assessment using Nmap and Wireshark.

## Tools Used
- Nmap
- Wireshark
- Windows 11
- Kali Linux

## Assessment Activities

1. Performed port scanning using Nmap.
2. Captured network traffic using Wireshark.
3. Analyzed DNS and TLS packets.
4. Reviewed network exposure.
5. Documented findings and recommendations.

## Deliverables

- Nmap Scan Results
- Wireshark Capture
- Network Security Assessment Report
- Screenshots

## Conclusion

The assessment identified the network services exposed during the scan and verified normal DNS and encrypted TLS traffic. The analysis indicates no unexpected open ports on the tested localhost system, demonstrating a limited attack surface.
